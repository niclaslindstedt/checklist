type: Changed
title: Cloud sync command centre
---

The cloud-sync glyph now always opens a redesigned details dialog — showing the backend and at-rest encryption state side by side, a one-tap Reload, and (in developer mode) a sync log (newest entries first, and now naming a dropped connection plainly instead of a cryptic "Load failed") — and tapping the glyph is the single, predictable way in whatever the sync state.
