title: Cloud sync command centre
---

The cloud-sync glyph now always opens a redesigned details dialog — showing the backend and at-rest encryption state side by side, a one-tap Reload, and an always-available sync log (which now names a dropped connection plainly instead of a cryptic "Load failed") you can read without turning on developer mode — and tapping the glyph is the single, predictable way in whatever the sync state.
