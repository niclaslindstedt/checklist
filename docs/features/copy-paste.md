# Copy and paste as markdown

Your [checklist](feature:checklist) travels as plain markdown, so it's easy to move a list out to another app — or pull one in. A header button copies the whole list to your clipboard, and the add-item field doubles as an import: paste a markdown checklist and each line lands as its own item.

## How it works

1. **Copy the whole list.** Tap the copy button in the header. The active list is written to your clipboard as a markdown task list — the list name as a heading, then a line per item with its box ticked or empty just as it stands. A confirmation toast and a brief tick let you know it copied.
2. **Paste a checklist to import.** Paste a markdown checklist straight into the add-item field. Instead of dropping in as a single line of text, each line is imported as its own item, appended to the current list.

When you import, the boxes carry over: items written as checked come in checked, and unchecked lines come in unchecked. Because the copy and paste sides speak the same markdown, a list you copy from one checklist pastes cleanly into another. Pasting ordinary text that isn't a checklist still behaves as a normal paste.
