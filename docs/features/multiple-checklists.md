# Multiple checklists

You don't have to cram everything into one list. `checklist` keeps **several
checklists side by side**, so you can give groceries, packing, and chores each
their own page and move between them without losing your place.

## How it works

1. Open the navigation menu to see all your checklists, listed by name.
2. **Switch** to any of them with a click — the one you're viewing is marked as
   active.
3. **Add a new one** from the same menu. New lists are named automatically —
   `Checklist`, then `Checklist 2`, and so on — so you can create one in a tap
   and name it properly later.
4. **Rename** the list you're on by clicking its title in the header and typing
   a new one.

Each list in the menu carries a **badge showing how many items it still has to
do** — the unchecked, not-yet-archived items — so you can see at a glance which
lists are finished and which still need attention. The badge disappears once a
list is empty or fully checked off.

Want even more separation than several lists in one place? Group your lists
into [namespaces](feature:namespaces). And when an item is done but you'd rather
not delete it, send it to the [archive](feature:archive) instead.
