# Item notes

Every item is more than a single line. Tap an item to edit its text in place,
and tuck a longer **note** beneath the title for the detail that doesn't belong
in the headline — a link, a few steps, the context you'll want later.

Notes are written in **markdown**, so `**bold**`, `*italic*`, `` `code` ``,
lists, and links all render the way you'd expect. While editing, the note
shows as plain text; once you close the editor it reads as formatted markdown.

## How it works

1. **Edit the text.** Tap an item to open it for editing in place. Type your
   change and press `Enter` to commit, or tap away.
2. **Add a note.** On desktop, press `Shift+Enter` to drop into the note field
   without leaving the keyboard; or tap **`Add a note`** in the editor. Write
   your note and commit.
3. **Reveal it.** An item that carries a note shows a **note glyph** beside its
   title — grey while the note is hidden, highlighted while it's open. Tap the
   glyph (or the title) to expand the note, rendered as markdown. Tap a link
   inside the note to follow it.
4. **Edit it again.** With the note revealed, tap the title to edit the title
   or tap the note body to edit the note. Tap outside the row to collapse it.

Clearing a note's text removes it, and the note glyph disappears with it — so an
item only carries the weight it needs.
